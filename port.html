MajorProject

Phishing Website Detection Using Machine Learning Algorithms

My main project is on phishing website detection using machine learning algorithms. Phishing refers to an attempt to steal sensitive information, typically in the form of usernames, passwords, credit card numbers, bank account information or other important data in order to utilize or sell the stolen information.

The story behind the name phishing?
Hackers generally use the letter “ph” instead of “f” and therefore initially they were known as phreaks.


I choose this project because it combines my passion for technology with real-world applications. It also allows me to explore areas like machine learning and software development simultaneously.Detection of phishing websites is performed using machine learning algorithms like logistic regression, Decision tree, random forest, Gradient Boosting. In the data collection phase, the data is collected on both phishing and legitimate websites. Then comes extracting the useful features of the given dataset.
"Legitimate websites are nothing but legal websites"
Logistic regression : Logistic regression is a supervised machine learning algorithm that accomplishes binary classification tasks by predicting the probability of an outcome, event, or observation.
Decision trees :  decision tree is a non-parametric supervised learning algorithm, which is utilized for both classification and regression tasks
Random forest : Random forests or random decision forests is an ensemble learning method for classification, regression and other tasks that operates by constructing a multitude of decision trees at training time.
Our team consists of four members including myself.Some challenges we faced include data collection and preprocessing, algorithm optimization, and integrating various modules seamlessly.Our project introduces innovative features such as personalized user recommendations based on machine learning algorithms and efficient route optimization, setting it apart from existing solutions.
